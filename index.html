Wplaceoki Discord OAuth2 Test

このサイトは、https://oauth2.wplaceoki.com を利用した Discord ロール認証の疎通確認用ページです。

GitHub Pages などの静的ホスティング上で公開し、 callback URL を KV に登録してテストします。

できること

  • Discord OAuth2 認証開始
  • callback の code / error 確認
  • /token 交換
  • /verify によるトークン確認

事前に必要なこと

  • test.html の URL を callback URL として使う
  • その URL を @ultimate_1113 に KV 登録依頼する
  • wplaceoki.com 配下のドメインで公開する
